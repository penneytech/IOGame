## How it works

- You write **Python** in a file called `character.py`.
- Your code lives inside one function: `build_character()`.
- That function returns a **dictionary** (a `{ ... }` block) that
  describes your character.
- The game runs your Python in your browser. The server only sees the
  final dictionary.

That's it. Click [Lesson 1 →](01-make-one.md) to start.
